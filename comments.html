HTML Fundamentals: Comments

<!-- HTML Comments -->

What it does:
HTML comments are NOT displayed in the browser. They are used to help document your code and explain what it does or perhaps what you did. I like to say comments explain code to the programmer and code explains comments to the computer. To write an HTML comment, you need to write it out just like the title. It goes as follows: <!-- Add your comment here -->

<!--Multi-Line
comment-->

What it does:
You can add multiple lines of text inside a multi-line comment. What's the difference you ask? NOTHING. As long as your comment is between this → <!-- ... -->, it will be commented. Easy right? I hope so!
