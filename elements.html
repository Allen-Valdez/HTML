HTML Fundamentals: Elements

HTML is comprised of elements, if we use these elements in a proper combination we can make an awesome website! but how do elements work and what exactly are they?

What are elements?

<> All elements have an opening and closing tag. The opening tag has the name of the element inside it and if they are NOT self closing then there is a closing tag with a slash before the name.
EXAMPLE:
  • Opening Tag → <p>
    • This opening tag displays a less than symbol, element name (In this example it is p for paragraph), and a greater than symbol. This completes the opening tag and will continue to put everything into a paragraph until you close it.
  • Closing Tag → </p>
    • The closing tag looks the same as the opening tag but NOTICE that the closing tag has a SLASH This completes the closing tag and will contain everything you wrote before it into the element.
Keep in mind that ALL elements close. Even if they are self closing tags. They will looks something like this → <img /> Notice the slash is AT THE END of the element before the greater than symbol.

Basic HTML elements

  • Header Elements
    • Different header elements are used to display different information. For example, the main heading, sub headings, and then sub headings for those sub headings!
    The header elements are: <h1>, <h2>, <h3>, <h4>, <h5>, <h6>.
  • Paragraph Element
    • A paragraph element is written using the <p> tag. This element is used to display text.
  • Horizontal Rule Element
    • A horizontal rule element is written using the <hr> tag. This element is used to display a horizontal line across the web page.
  • Anchor tag
    • An anchor tag is written using the <a> tag. This element is used to link to another page.
  • Ordered/Unordered List and List Item Elements
    • An ordered list is written using the <ol> tag. This element is used to display a list of items in NUMBERED order. An unordered list is written using the <ul> tag. This element is used to display a list of items in a bullet fashion. A list item is written using the <li> tag. This element is used to display a list item. You will use the list item element in both ordered and unordered lists.
  • Image Element
    • An image element is written using the <img> tag. This element is used to display an image. It also has an attribute called "src" which is used to add the source of that image. It can be from a local file or from a web server.
  • Divider Element
    • The divider element is written using the <div> tag. This element is used to containerize elements or "box up" certain areas in your body.
  • Span Element
    • A span is written using the <span> tag. Similiar to a div tag, however this element is more inline.
  • Table
    • The table element is written using the <table> tag. This element is used to represent tabular data. The table element displays data in a two-dimensional table made up of rows and columns.
  • thead
    • The thead element is written using the <thead> tag. This element is used to display the header of a table.
  • tr
    • The tr element is written using the <tr> tag. This element defines a row within the table element. The row's cells can be established using a mix of th and td elements.
  • th
    • The th element is written using the <th> tag. This element is used to define the header of a group of table cells. The th element is used inside the tr element.
  • tbody
    • The tbody element is written using the <tbody> tag. This element encapsulates a set of table rows indicating that they are the body of the table.
  • td
    • The td element is written using the <td> tag. This element is used to define a cell of a table indicating it is a table data.
  • tfoot
    • The tfoot element is written using the <tfoot> tag. This element is used to display the footer of a table. In other words, it sumarized the columns of the table.
