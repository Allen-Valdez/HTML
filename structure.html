HTML Fundamentals: Structure

When it comes to HTML, we want to know how to structure our page. This is very important because we want to be able to have a neat and beautiful page while also maintaining proper content and placing it in the proper place. It also helps with screen readers and other people who have disabilities.

<!DOCTYPE html>

What it does:
This is the first thing that comes before the <html> tag. It tells the browser what version of HTML we are using. In this case, we are using HTML5.

<html>

What it does:
This is the first thing that comes after the <!DOCTYPE html> tag. This is the root element of an HTML page. All other elements MUST be descendants of this element.

<head>

What it does:
This is the first thing that comes after the <html> tag. This element contains all the metadata (machine-readable information) about the page. This includes the title, the scripts/style sheets, and keywords that will help with the SEO (Search Engine Optimization).

<title>

What it does:
This element is the broswer's title or the page's tab. It only contains text and all tags with within the element will be ignored.

<body>

What it does: This elements contains ALL the content you wish to display on the web broswer. There can only be one <body> element per page. Again, everything you want the user to see goes here.
