HTML Fundamentals: Attributes

Attributes are extra information about the element that you will not see in the actual content. All HTML elements can have attributes and they are always specified in the opening tag. They usually come in name/value pairs like → name="value".

Now let's look at an example using a paragraph tag → <p attributename="value">Some Content</p>

Easy right? Now lets go over some other attributes

  • The id Attribute
    • The "id" attribute is written like this: id="idName"
  • The class Attribute
    • The "class" attribute is written like this: class="className".
